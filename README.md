# Restaurant POS System

A modern, full-stack restaurant point-of-sale system built with React, Node.js, and PostgreSQL.

## Features

- **Real-time Order Management**: Live order updates across all devices using WebSocket
- **Kitchen Display System**: Dedicated view for kitchen staff to manage orders
- **Table Management**: Interactive floor plan with drag-and-drop table positioning
- **Payment Processing**: Integrated payment handling with split bill support
- **Menu Management**: Dynamic menu with categories, modifiers, and inventory tracking
- **Employee Management**: Role-based access control and clock-in/out tracking
- **Analytics Dashboard**: Sales reports, inventory insights, and performance metrics

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Zustand
- **Backend**: Node.js, Express, TypeScript, Prisma ORM
- **Database**: PostgreSQL
- **Real-time**: Socket.IO
- **Monorepo**: Turborepo
